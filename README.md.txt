About This Project
customer complaints management is made of a combination of modules that work in collaboration with each other and make it beneficial to accomplish the main aim of the scheme. This interface is made for users who are facing any problem or issue and want to make a complaint to get rid of it. The interface allows them to give a description of the problem, i.e. when they encountered the problem and where it was.
